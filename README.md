# Experiment-4-ADDITION-OF-2-NUMBERS
Aim

To create an Android application that performs addition of two numbers entered by the user.

Algorithm

1. Open Android Studio.


2. Create a new Android project.


3. Open activity_main.xml.


4. Add two EditText fields to enter numbers.


5. Add an Addition button to perform calculation.


6. Add a Reset button to clear inputs.


7. Add a TextView to display the result.


8. Open MainActivity.java.


9. Declare EditText, Button and TextView variables.


10. Use findViewById() to connect UI components.


11. Set OnClickListener for the Addition button.


12. Read numbers entered by the user.


13. Convert text values to numeric values.


14. Add the two numbers.


15. Display the result in the TextView.


16. Set OnClickListener for Reset button.


17. Clear both input fields and result.


18. Run the application and verify the addition operation.
